This dashboard shows code examples and mini experiments. You will run the code on your own machine, then come back here to compare and explore.

Step 1: Install Python

Use Python 3.10 or 3.11. These are stable, widely supported versions and are a safe choice for beginners. Install from the official Python website.

During installation, make sure "Add Python to PATH" is checked. This lets you run python from any terminal.

Step 2: Create a project folder

Create a folder on your computer for this learning project. Keep all your .py files here.

Step 3: Create a virtual environment

A virtual environment keeps project packages isolated from the rest of your system. This prevents version conflicts and makes projects easier to reproduce later.

python -m venv venv

This creates a separate local Python environment inside a folder called venv.

Step 4: Activate the virtual environment

Windows:

venv\Scripts\activate

Mac / Linux:

source venv/bin/activate

Your terminal prompt should change to show (venv).

Step 5: Install Python packages

Install packages only inside the venv. This keeps project dependencies separate from other Python projects on the same computer.

pip install numpy pandas scikit-learn torch

You can install matplotlib later if you want to plot things.

Step 6: Run code

Save snippets into .py files and run them with:

python file.py

Or use interactive Python for quick experiments:

python

Step 7: Use VS Code

VS Code is a simple, beginner-friendly editor with a built-in terminal and good Python support. It makes it easier to edit, run, and debug scripts. Download it from code.visualstudio.com.

Step 8: Special note for the RNA-seq section

The Python RNA-seq module (Module 9) will create a file called rna_seq.csv in your project folder. The R module (Module 10) reads that CSV file. Keep both scripts in the same folder.

R, RStudio, and Bioconductor

R is used for the final RNA-seq analysis module. RStudio is the easiest way to work with R because it gives you a console, editor, plotting tools, history, debugging, and workspace management in one place. Bioconductor is the main bioinformatics ecosystem for R and is the right place to look for genomics and RNA-seq packages.

Step 1: Install R

Install the latest stable R from CRAN. RStudio depends on R, so R must be installed first.

Step 2: Install RStudio Desktop

RStudio is the recommended IDE for R. It works on Windows, macOS, and Linux. Download from posit.co. Open RStudio after installing R.

Step 3: Learn the basic R workflow

R can be used from the console directly or through RStudio. Here are the essentials:

  • <- is the normal assignment operator in R
  • c() creates vectors
  • data.frame() creates tabular data
  • read.csv() reads CSV files
  • library() loads packages
  • help.start() opens the help browser
  • ?mean shows help for a function
  • getwd() shows the current working directory
  • setwd() changes the working directory

Step 4: Install Bioconductor support

Bioconductor packages are usually installed through BiocManager. Run these in R or RStudio:

if (!requireNamespace("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install(version = "3.22")

The current Bioconductor release is version 3.22 and it works with R 4.5.0. If the release changes later, only this line needs updating.

Step 5: Install example Bioconductor packages

BiocManager::install(c("GenomicFeatures", "AnnotationDbi", "Biostrings"))

Step 6: Use Bioconductor packages

After installation, packages are loaded with library(PackageName). Bioconductor is especially useful for genomics, transcriptomics, annotation, and RNA-seq workflows.

Step 7: Special note for the RNA-seq module

The Python module writes rna_seq.csv. The R module reads that file inside RStudio or R. Keep both files in the same folder or set the working directory appropriately with setwd().
